## Getting Started

To get a local copy up and running follow these simple example steps.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/chadow4/Snake-Game.git
   ```
2. Run the game on a local/external server as we use JQuery for parsing json files.
